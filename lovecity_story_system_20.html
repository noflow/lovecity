📦 How to get your dialogue into the game

  1. Build your scene(s) using the Story Arcs tab
  2. Click "🚀 Export to Game Files" below
  3. Download the generated .js file(s)
  4. Drop them into lovecity/src/dialogue/npcs/
  5. If it's a new character, add one import line to src/dialogue/index.js
  6. Done — no other changes needed!
